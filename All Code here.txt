===============Outline CSS System==================

Outline CSS- এ লিখার জন্য আমাদের <head> </head> Tag - এর ভেতর সব কিছু লিখতে হবে।

<title> </title> Tag- এর পরে লিখলেই বেশি সুন্দর হয়। 

CSS এ লিখা ডিজাইন করার জন্য অবশ্যই- <title> </title> Tag- এর পরে আরেকটি ট্যাগ ব্যবহার করতে হয়।
সেটি হলো- <style> </style> Tag.............

করে দিতে হবে। আর এই জন্যই তার জন্য দরকার---- Identifier------

1. Tag.
2. Class.
3. ID.

bg      = Background
small   = Small
md      = Medium
md      = Margin Bottom
mt      = Margin Right

1. success  = Green type
2. Info     = Blue type
3. Dark     = Black or Ash
4. Light    = White type
5. Warning  = Golden or Yellow type
6. Danger   = Red type
7. Primary  = Blue type
8. Secondary= Ash type


Device Name : For Responsive Device

small   = Small
md      = Medium
lg      = Large
xl      = Extra Large
